# PeerChat

Peer-to-Peer messenger written in SwiftUI, using [Multipeer Connectivity](https://developer.apple.com/documentation/multipeerconnectivity)

<img src="https://github.com/Lukas1h/PeerChat/assets/53445584/7c19de04-4e48-4dfa-b900-013699501928" height="500">

## Supported platforms
- iOS/iPadOS 17+
- macOS 14+

## Featues
- No Wi-Fi needed
- Supports emojis, voice messages & files
- *Annnnnnnd more!*
